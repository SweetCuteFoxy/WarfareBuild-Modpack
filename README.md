# WarfareBuild Modpack

Minecraft 1.20.1 Forge modpack (~296 mods) with military/tactical theme and custom WarZone mod.

## Download

Go to [Releases](../../releases) and download all 4 parts of WarfareBuild.zip.

### Combine parts

**Windows (CMD):**
`
copy /b WarfareBuild.zip.part01+WarfareBuild.zip.part02+WarfareBuild.zip.part03+WarfareBuild.zip.part04 WarfareBuild.zip
`

**Linux/macOS:**
`ash
cat WarfareBuild.zip.part01 WarfareBuild.zip.part02 WarfareBuild.zip.part03 WarfareBuild.zip.part04 > WarfareBuild.zip
`

## Install

1. Open **Prism Launcher**
2. Click **Add Instance** -> **Import**
3. Select the combined `WarfareBuild.zip`
4. Wait for import to finish
5. Launch!
